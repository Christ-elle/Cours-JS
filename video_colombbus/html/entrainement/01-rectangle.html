Rectangle : calculer périmètre / aire d'un rectangle

Objectifs

Ecrire un script qui affichera dans la console les valeurs choisies pour la largeur et la longueur d'un rectange puis le périmètre et l'aire d'un rectangle

A faire

  • Créer un script js/entrainement/01-rectangle.js
  • La largeur sera stockée dans une variable largeur; idem pour la longueur
  • Le script affichera dans la console :
    • Paramètres du rectangle : longueur 7, largeur 9
    • L'aire du rectangle est 63
    • Le périmètre du rectangle est : 32
    • Ajouter un lien vers le script fini

Pour aller plus loin

  • Chercher comment utiliser une formule avec une multiplication pour le périmètre
Aide
  • Créer les variables en utilisant let bob
  • Assigner immédiatement leur valeur choisie au hasard avec le signe =
  • Pour conserver les types des éléments affichés dans la console, utiliser console.log() avec plusieurs paramètres, donc séparés par des virgules
Notions à acquérir
  • Créer de script dans un fichier séparé
  • Différences entre <script> et un fichier séparé
  • Création de variable type number
  • Différences type number et le type string
  • Assigner une valeur à une variable
  • Afficher le contenu d'une variable
  • Afficher plusieurs paramètres
  • Conserver le type de plusieurs paramètres
