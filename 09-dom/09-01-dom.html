Définition

1- Le DOM est une interface de programmation pour les documents HTML ou XML qui représente le document ( la page web actuelle ) sous une forme qui permet au langage de script comme le JavaScript d'y accéder et d'en manipuler le contenu et les styles.

2- Les objets node ou les nœuds du DOM

Le terme nœud est un terme générique qui sert à désigner tous les objets contenus dans le DOM.

À partir du nœud racine html, on voit que 2 branches se forment

3- Alors que JavaScript est le langage de programmation qui permet d'opérer sur les objets DOM et de les manipuler, le DOM fournira les méthodes et les propriétés nécessaires pour lire, modifier, mettre à jour et supprimer des éléments du document examiné. Par exemple, vous pouvez récupérer la valeur d'un champ texte HTML dans une chaîne à l'aide du DOM.
