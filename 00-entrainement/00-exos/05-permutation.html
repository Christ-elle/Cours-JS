Permutation de 2 variables

Ecrivez un programme '05-permutation.js' ayant initialement le ocntenu suivant.
var nombre1 = 5;
var nombre2 = 3;

Console.log(nombre1);// Doit afficher 3

Console.log(nombre2);// Doit afficher 5

*A l'endroit indiqué par un commentaire, ajoutez le code necessaire pour inverser les valeurs des 2 variables.

Il existe plusieurs solutions à cet excercice.
Astuce : vous n'êtes pas limité à l'utilisation de deux variables
