TVA

Ecrivez un programme JS qui fait saisir un prix HT (hors taxes) à l'utilisateur, puis affichez le prix TTC (toutes taxtes comprises) correspondant en se basant sur un taux de TVA à 20%


Vous pouvez effectuer l'affichage dans la console, dans une pop-up ou sur la page directement, dans un paragraphe avec un id

Indice : Multiplier le prix HT par 1,20 pour obtenir le prix TTC
