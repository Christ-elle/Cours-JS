Valeurs



Observez le code ci-dessous et tentez de prévoir lesvaleurs finales de chaque variable à la fin de chaque ligne ( par exemple après la ligne 'var a=2' si je faisais un console.log dea j'aurai 2 comme résultat



L'idée est de faire vos calculs vous même sur une feuille avant de regarder la console.

  • var a = 2;
  • a = a - 1;
  • a++;
  • var b = 8;
  • b += 2;
  • var c = a + b * b;
  • var d = a * b + b;
  • var e = a * (b + b);
  • var f = a * b / a;
  • var g = b / a * a;
