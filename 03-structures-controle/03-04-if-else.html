1- if.... else...if... else en JS

La condition if... else (si... sinon si... sinon) est une structure conditionnelle encore plus complète puisqu'elle permet de générer et de prendre en charge autant de cas que l'on souhaite.


Nous allons pouvoir ajouter zutant de else if que l'on veut dans cette condition.

let x = 0.5; if(x > 1){ alert('x contient une valeur strictement suppérieure à 1'); }else if(x==1){ alert('x contient la valeur1'); }else { alert('xcontient une valeur strictement inférieure à 1'); }

Dès qu'un test va être validé le JS va ignorer les tests suivants, il faut donc faire bien attention à l'ordre des else if lors de l'écritire d'une condition pour obtenir le résultat souhaité.
