JS- III 01 Les structures de contrôle


La condition if est l'une des plus utilisée et est également la plus simple à apréhender puisqu'elle va nous permettred'excecuter un bloc de code SI et seulement SI le résultat vaut true

2- if éxécutés dans le if.js

  • (x > 1){ alert('x contient une valeur supérieur à 1');
    }
  • (x == y){ alert('x et y contiennent la même valeur');
    }
  • if(y){ alert('La valeur de y est évalué à true');
    }

!! Dans ce 3e if: tout test d'une condition va petre évalué dans un contexte booléen; donc à trueà l'éxception des valeurs suivantes qui vont être évaluées à false.

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

2- Inverser la valeur logique d'un test.

Dans un if le code placé dans le if n'est exécuté que si le résultat de la comparaison est true.

Dans certaines situations, on préfère vérifier si la comparaison est false.

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false.
