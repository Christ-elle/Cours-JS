JS- III 01 Les structures de contrôle


1- Conditions et les boucles

Il existe deux types de structures de contrôle de base que l'on retrouve dans tous les languages de programmation

  • Les structures de contrôle conditionnelles: Les Condition
  • Les structures de contrôle de boucles Boucles

Les conditions vont nous permettre d'excecuter un certain nombre d'instruction SI et seulement SI une certaine condition est véréifiée.

Les boucles vont nous permettre d'excecuter un bloc de code en boucle tant qu'une condition donnée est vérifiée.


A- Présentation des conditions

  • La condition if(si) ;
  • La condition if...else(si...sinon) ;
  • La condition if....elseif...else (si...sinonsi...sinon).
