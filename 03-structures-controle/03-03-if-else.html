1- if.... else... en JS

La structure conditionnelle if ... else ( en français si ... sinon) sera plus complète car on va exécuter un bloc de code si un test renvoie trueou un autre dans le cas contraire.

If .. else...

  • if(x > 1){
    alert('x contient une valeur supérieure à 1');
    }elsef{
    alert('x contient une valeur inférieure à 1');
    }
