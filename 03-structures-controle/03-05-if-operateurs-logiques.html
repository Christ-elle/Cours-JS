JS- III 05 OPERATEURS LOGIQUES


1-

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dasn nos tests ou d'inverser la valeure logique du résultat d'un test.

Présentation et utilisation des opérateurs logiques.

Le JS supporte 3 opérateurs logiques : L'opérateur logique "ET", "OU" et "NON".

OPERATEURS LOGIQUES
Opérateurrs,nom opérateur symbole Déscription
AND(ET) && Renvoie true si toutes les comparaisons sont évaluées à true, ou false dans le cas inverse.
OR(OU) || Renvoie true, si au moins l'ine des comparaisons est évaluée à true ou false sinon.
NO(NON) ! Renvoie false si une comparaison est évaluée à true, ou renvoie true dans le cas contraire.
